# 股票监控系统 - Stock Watch

## 目标
帮助用户在 A 股市场赚钱，提供：
- 实时监控
- 异动预警
- 数据分析
- 盘后复盘

## 功能模块

### 1. 数据获取
- 实时行情（价格、成交量、涨跌幅）
- 财经新闻
- 公司公告
- 行业数据

### 2. 监控预警
- 涨跌幅超过阈值
- 成交量异常
- 重要新闻/公告

### 3. 分析报告
- 盘中定时汇报
- 盘后总结
- 周/月复盘
