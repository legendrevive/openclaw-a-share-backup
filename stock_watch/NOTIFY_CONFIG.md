# 📱 飞书通知配置说明

## ✅ 已启用通知类型

| 通知类型 | 触发条件 | 发送时间 |
|---------|---------|---------|
| 🚨 **价格警报** | 涨跌幅≥5% 或达到目标价 | 实时 |

---

## 🎯 价格警报说明

### 自动触发条件

1. **大幅波动**: 两次检查间隔涨跌超过 5%
2. **目标价提醒**: 达到设定的目标价格

### 设置目标价

编辑 `stocks.json`:

```json
{
  "name": "洛阳钼业",
  "code": "603993",
  "market": "SH",
  "target_price_up": 25.00,    // 涨到 25 元提醒
  "target_price_down": 22.00    // 跌到 22 元提醒
}
```
